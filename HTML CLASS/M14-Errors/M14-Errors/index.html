Wonky Site

This is a test site. I'm written a little wonky though.

This is a simple site with a few errors in it. Some you can clearly see in the browser here. Some you'll have to go looking through the code to find. Your job is to try and find all the errors. As a web developer, it's a very important skill to be able to read and troubleshoot code. This assignment will let you practice that. The validator at validator.nu will catch a lot but it won't catch all of them so don't rely entirely on it. If you use the validator, the warnings about adding more headings can be ignored. But there will be other warnings like lang="en" and missing alt text you should pay attention to. Keep track of the changes you make by editting the Error Tracker.
